- 게임 이름
이세계에서 게임잼

- 장르
시뮬레이션 / 타이쿤

- 플랫폼
윈도우

- 플레이 타임
10분 내외

- 제작 인원
그래픽 1인, 기획 3인, 프로그래밍 2인
담당 파트: 프로그래밍

- 게임 소개
충북 게임 아카데미에서 수업을 듣던 수강생들이 갑자기 이세계로 들어가면서 게임잼에 참여하게 된다.
게임 속에서 게임을 제작한다.
장르를 잘 조합하고, 미니 게임들을 플레이하면서 높은 점수를 얻어 최종 점수를 높게 받는 것이 목표이다.

- 게임 방법
마우스와 키보드를 이용해 미니게임과 각종 이벤트를 플레이한다.


담당 
미니 게임: CatchBug, JigsawPuzzle
DataManager: PlayerPref을 이용해 데이터 저장 구현,
MainGame:  StartTalk, TextEvent. 구현
UI: MouseCursor, EventManager(Thema 관련 구현)
Character: Stats 구현
